Centered heading

To center a heading in HTML, you can simply use the style attribute with the text-align:center; property. This tells the browser to align the heading text in the middle of the page. For example: <h2 style="text-align:center;">Centered Heading</h2>. It’s an easy and it only changes how text is displayed—it doesn’t hide or overlap any elements. Always keep your text readable and well-spaced for a clean, professional look.

Mark tag

The <mark> tag in HTML is used to highlight or emphasize important text within a paragraph. When applied, it gives the selected text a yellow background by default, helping readers notice key terms or phrases. For example: <p>HTML stands for <mark>HyperText Markup Language</mark>.</p> will display the words “HyperText Markup Language” with a highlight effect. This tag is completely improves readability and does not hide or mislead content. Use it to draw attention to definitions, keywords, or short notes in your articles.

Bold text

In HTML, you can make text bold using the <b> or <strong> tag. The <b> tag simply makes the text bold for visual emphasis, while the <strong> tag also adds importance to the text, which helps with accessibility and SEO. For example: <p>This is <b>bold</b> text.</p> or <p>This is <strong>important</strong> text.</p>. They only change the style of the text and do not hide or manipulate content. Use bold text to highlight key points or headings so your content is easier to read.

A Description List

A description list in HTML is used to display a list of terms and their corresponding descriptions. It helps organize definitions or key-value information in a clean and structured way. A description list is created using the <dl> tag, along with <dt> for the term and <dd> for its description.

Article tag and Header tag

The <article> tag in HTML is used to define a self-contained piece of content, such as a blog post, news story, or tutorial section. Each article should make sense on its own and can include headings, paragraphs, images, or links. The <header> tag, on the other hand, is used to group introductory content or navigation links at the top of a page or article. For example: <article><header><h2>My First Post</h2></header><p>This is my article.</p></article>. Both tags improve the structure and meaning of your webpage, making it easier for search engines to understand.

Article tag and Image

The <article> tag in HTML is used to define a self-contained piece of content, such as a blog post, news item, or tutorial. Inside an article, you can include headings, paragraphs, lists, and images to make the content informative and engaging. The <img src> tag is used to display images on your webpage. The src attribute specifies the image file’s path, and you can add alt text to describe the image for accessibility. For example: <article><h2>My Science Experiment</h2><img src="experiment.jpg" alt="Science experiment"><p>This experiment shows how plants grow.</p></article>.

Aside tag

The <aside> tag in HTML is used to define content that is related to the main content but separate from it, such as side notes, advertisements, or related links. It is commonly used for sidebars, callouts, or additional information that complements the main article. For example: <aside><p>Related topic: How plants perform photosynthesis.</p></aside>.

Back to top link

A "Back to Top" link in HTML allows users to quickly return to the top of a webpage without scrolling manually. You can create this feature using an anchor link. First, add an id to the top element, like <h1 id="top">My Page</h1>, and then create a link pointing to that ID: <a href="#top">Back to Top</a>.

Div tag

The <div> tag in HTML is a container element used to group other HTML elements together. It does not add any visual effect on its own but is very useful for organizing content and applying styles with CSS. For example, you can wrap a heading, paragraph, and image inside a <div> to style them as a single section: <div style="background:#f5f5f5; padding:10px;"><h2>My Section</h2><p>This is a paragraph.</p></div>.

Footer tag

The <footer> tag in HTML is used to define the footer section of a webpage or a specific section of content. It typically contains information like copyright notices, contact details, links to privacy policies, or navigation links. For example: <footer><p>© 2025 LearnScienceXY. All rights reserved.</p></footer>.

Backgroung colour, font size , font family , font colour

In HTML, you can customize the appearance of your webpage by changing the background color, font size, font family, and font color using the style attribute or CSS. For example, to set a light background with readable text, you can use: <body style="background-color:#f0f0f0; font-family:Arial, sans-serif; font-size:16px; color:#333;">. Here, background-color sets the page’s background, font-family defines the text style, font-size controls the size of the text, and color sets the font color.

Rowspan

In HTML tables, the <th> tag is used to define table headers, and the rowspan attribute allows a header cell to span multiple rows. This is useful when you want a single header to cover several rows of related data, making the table easier to read. For example: <th rowspan="2">Name</th> will merge the header cell across two rows.

Superscripted and Subscripted tag

In HTML, the <sup> tag is used to display text as superscript, which appears slightly above the normal text line. The <sub> tag displays text as subscript, appearing slightly below the normal text line. These tags are commonly used for mathematical expressions, chemical formulas, and footnotes. For example: H<sub>2</sub>O displays the chemical formula for water, and x<sup>2</sup> represents x squared.

tr and td tag

In HTML, the <tr> tag is used to define a table row, and the <td> tag defines a table cell within that row. Together, they allow you to create structured tables for displaying data in rows and columns. For example: <tr><td>Name</td><td>Age</td></tr> creates a row with two cells labeled "Name" and "Age."

Table width

In HTML, you can make a table stretch across the full width of its container by using the width="100%" attribute or CSS styling. This ensures the table automatically adjusts to different screen sizes, making it more readable and responsive. For example: <table style="width:100%; border:1px solid #333;"></table> creates a table that spans the entire width of the page with a visible border.

Table border, bg colour

In HTML, you can enhance the appearance of tables by adding a border and a background color. The border attribute or CSS adds visible lines around the table and its cells, while background-color sets the table’s background for better readability. For example: <

Table row height

In HTML, you can control the height of table rows to make your tables more readable and visually appealing. This can be done using the height attribute in the <tr> tag or through CSS styling. For example: <tr style="height:50px;"><td>Content</td></tr> creates a table row with a height of 50 pixels.

Colspan

In HTML tables, the colspan attribute is used to make a single table cell span across multiple columns. This is helpful when you want to create a header or section that covers more than one column for better table organization. For example: <td colspan="2">Merged Cell</td> will combine two columns into one cell.

Nav tag

The <nav> tag in HTML is used to define a section of navigation links on a webpage. It helps organize menus, site links, or table-of-contents links in a semantic and structured way. For example: <nav><a href='index.html'>Home</a> | <a href='about.html'>About</a></nav> creates a simple navigation menu.

Ordered list

In HTML, the <ol> tag is used to create an ordered (numbered) list, while the <li> tag defines each item within that list. Ordered lists are useful for instructions, steps, or ranking information. For example: <ol><li>Step 1</li><li>Step 2</li></ol> will display a numbered list with two steps.

Ordered list with letters

In HTML, you can create an ordered list that uses letters instead of numbers by using the type attribute with the <ol> tag. For example, <ol type="A"><li>Item 1</li><li>Item 2</li></ol> will display a list with items labeled as "A", "B", and so on. This method is useful for outlining points, options, or sections in a structured way.

Section tag

The <section> tag in HTML is used to define a distinct section of a webpage that groups related content together. Each section can include headings, paragraphs, images, and other HTML elements to create a meaningful block of information. For example: <section><h2>About Our Website</h2><p>LearnScienceXY provides tutorials and articles.</p></section>
